# E-Commerce Customer Churn Analysis (MySQL)
## Project Overview

This project analyzes customer churn behavior in an e-commerce platform using SQL.
The goal is to identify patterns and factors influencing customer attrition.

## Objectives
Perform data cleaning (handling NULL values and outliers)
Standardize inconsistent data
Transform dataset using new columns
Analyze churn patterns using SQL queries

## Tools Used
MySQL
SQL (Joins, Aggregations, Subqueries)

## Key Insights
Identified churned vs active customers
Analyzed customer behavior based on payment mode, tenure, and satisfaction
Categorized customers based on distance from warehouse

## Features
Data Cleaning (Mean & Mode Imputation)
Data Transformation
Business Analysis Queries
Customer Returns Analysis

## Files Included
customer_churn.sql – Database creation and dataset
analysis_queries.sql – All SQL queries used in analysis

## Conclusion
This project demonstrates how SQL can be used to derive meaningful business insights and improve customer retention strategies
